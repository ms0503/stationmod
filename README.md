# Station MOD / 駅MOD
## Description
This is a mod for Minecraft 1.12.2(Forge #2860).   
It adds many station-related items, such as rail and ticket.

駅MODはMinecraft 1.12.2(Forge #2860)で動作するMODです。  
このMODはレールや切符などの駅に関する物を追加します。
